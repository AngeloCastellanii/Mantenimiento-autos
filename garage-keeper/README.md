# GarageKeeper

Demo de mantenimiento vehicular — React + Mantine.

**Demo en vivo:** https://mantenimiento-autos.vercel.app

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy (Vercel)

- Root directory: repo root (usa `vercel.json` en la raíz) **o** `garage-keeper`
- Build: `pnpm build`
- Output: `dist`
- Install: `pnpm install`

## Flujo de la app

1. **Inicio** — resumen, alertas y accesos rápidos  
2. **Vehículos** — registra tus autos  
3. **Detalle** — historial de servicios por vehículo  
4. **Próximos** — alertas por km o tiempo
